## 🇬🇧 Hacklily Desktop (English)

**Hacklily Desktop** is a macOS and Windows desktop application for editing and rendering scores written in [LilyPond](http://lilypond.org/) **entirely offline**. Rendering is done locally, with no dependency on any external server.

> Prefer the web version? The original project runs online at **[hacklily.org](https://www.hacklily.org)** — no installation needed.

### ⚠️ macOS: First Launch (App Not Notarized)

Because Hacklily Desktop is not notarized with Apple, macOS Gatekeeper will block it the first time you try to open it and show a message saying the app is "damaged" or "cannot be opened".

**To open the app, follow these steps:**

1. Copy the app to your **Applications** folder (drag it from the `.dmg`).
2. Open **Terminal** (you can find it in Applications → Utilities).
3. Run the following command:

```bash
xattr -cr "/Applications/Hacklily Desktop.app"
```

4. Now double-click the app normally — it will open without any warning.

> This command removes the quarantine flag that macOS sets on apps downloaded from the internet that are not notarized. It is safe to run.

### ✨ Features

#### File Menu (`File ▾`)

| Option | Description |
|--------|-------------|
| **New song** | Creates a new empty score from a starter template. |
| **Open…** | Opens a local `.ly` file from your disk. |
| **Import → Browse MutopiaProject…** | Opens the integrated MutopiaProject browser. Navigate thousands of free LilyPond scores, click on any one to load it instantly — including multi-file scores (quartets, orchestra…), which are assembled automatically. |
| **Import → Import MusicXML…** | Converts a MusicXML file (`.xml`, `.mxl`) to LilyPond source and loads it into the editor. |
| **Import → Import MIDI…** | *(Desktop only)* Converts a MIDI file to LilyPond source using `midi2ly` and loads it into the editor. |
| **Save** | Saves the current file back to disk (only available when a local file is open). |
| **Save as…** | Saves the current score to a new local `.ly` file. |
| **Export → LilyPond source** | Exports the source code as a `.ly` file. |
| **Export → PDF** | Exports the rendered score as PDF. |
| **Export → SVG** | Exports the rendered score as SVG (vector). |
| **Export → PNG** | Exports the rendered score as PNG (raster image). |
| **Export → MIDI** | Exports the MIDI data generated by LilyPond. |
| **Use light / dark colour scheme** | Toggles between light and dark editor theme. |
| **LilyPond manual…** | Opens the official LilyPond documentation in the browser. |
| **Check for updates…** | Opens the GitHub releases page to check for a newer version. |
| **Source code…** | Opens the source code repository on GitHub. |
| **About Hacklily Desktop** | Shows version and authorship information. |

#### Insert Menu (`Insert ▾`)

Assistants for inserting common LilyPond elements into the editor at the cursor position.

| Option | Description |
|--------|-------------|
| **Clef…** | Interactive selector to insert a clef (`\clef`). |
| **Key Signature…** | Interactive selector to insert a key signature (`\key`). |
| **Time Signature…** | Interactive selector to insert a time signature (`\time`). |
| **Notes…** | Interactive note picker to insert note sequences. |

#### Snippets (`Snippets`)

A library of reusable LilyPond code fragments.

| Tab | Description |
|-----|-------------|
| **LilyPond Official** | Browse all official LilyPond snippets from [lilypond.org](https://lilypond.org/doc/v2.24/Documentation/snippets-big-page.html), organized by category. Requires an internet connection to load. Click **Load** on any snippet to insert it into the editor. |
| **My Snippets** | Save the current editor code as a personal snippet (with name and category). Load or delete your saved snippets at any time. Snippets are stored locally on your machine. |

#### Playback Controls

Once LilyPond has generated MIDI output (your score must include a `\midi {}` block inside `\score {}`), the play button activates:

| Control | Description |
|---------|-------------|
| **Play / Pause** | Plays or pauses MIDI playback using sampled instruments. |
| **Rewind** | Steps back 4 seconds. |
| **Fast-forward** | Steps forward 4 seconds. |
| **Time display** | Shows current playback position in seconds. |

#### Editor

- **Monaco Editor** (the same engine that powers VS Code) with LilyPond syntax highlighting and autocompletion.
- **100% local rendering** — LilyPond runs on your machine, no internet required for rendering.
- **Split view** — editor and score preview side by side; on narrow windows, switch between edit-only and view-only mode.
